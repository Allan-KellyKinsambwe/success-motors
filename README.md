## Tech Stack
- Flutter (Dart)
- Firebase Authentication & Firestore
- Video Player (optional onboarding upgrade)
- Java 17 for Android builds
- Smooth Page Indicator, Google Fonts, etc.

## Setup & Run
```bash
# Clone the repo
git clone https://github.com/Allan-KellyKinsambwe/success-motors.git
cd success-motors

# Install dependencies
flutter pub get

# Run on device/emulator
flutter run
